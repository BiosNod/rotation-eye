### AI
Проект представляет собой тест на математические способности модели через простейшую задачу по расчёту углов вращения одного объекта относительно другого.

### Информация:
Есть точка XYZ (камера) в пространстве и точка глаза (XYZ), в реалтайме угол поворота (rotation) глаза (RX, RY, RZ) расчитывается через lookAt чтобы он смотрел в точку XYZ камеры. Т.е. при перемещении камеры глаз смотрит прямо в неё.
- Система координат — стандартная THREE.js: Y-up, Z-forward (камера смотрит по -Z)
- Позиция глаза  и позиция камеры прописаны в коде
- Камера управляется OrbitControls (мышкой)

### Задача:
Написать свою идентичную функцию, которая сделает расчет поворота глаза чтобы не использовать lookAt и сделать в UI переключение между lookAt и своей новой функцией расчета поворота lookAtCustom и чтобы поведение поворота было идентичным.

### Запрещается:
Использовать готовые классы для расчетов из THREE (любые), например, .normalize (сам пиши код нормализации), THREE.Quaternion, THREE.Vector3 и др., сами кватернионы и векторы использовать разрешается, но только если ты вручную сам напишешь код для их расчёта без использования готовых функций/классов
