# 👁️ Rotation Eye

Проект для проверки математических способностей ИИ.  
Задача — реализовать расчёт вращения 3D-объекта **без использования готовых функций**.

---

## 🛠️ Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/BiosNod/rotation-eye.git
cd rotation-eye

# 2. Установить зависимости
npm install

# 3. Запустить приложение
npm start
```

---

## 📖 Описание задачи

В проекте есть:
- **Глаз** — 3D-модель, которая должна поворачиваться
- **Камера** — точка наблюдения, которую пользователь перемещает мышкой

### 🔸 Что делает встроенная функция `lookAt`

В библиотеке **Three.js** есть встроенный метод `object.lookAt(position)`.  
Он **автоматически поворачивает объект** так, чтобы он смотрел в указанную точку.

В коде это выглядит так:
```javascript
eye.lookAt(camera.position);
```

### 🔸 Что нужно сделать ИИ

Написать дополнительно **свою собственную функцию** с тем же поведением как и у `eye.lookAt()`:
```javascript
lookAtCustom(eye, camera.position);
```

Функция должна вычислять углы вращения `(rotation.x, rotation.y, rotation.z)` глаза так, чтобы он смотрел на камеру так же, как и при вызове `eye.lookAt()` как это сделано сейчас в коде.

## ⚙️ Требования

| ✅ Можно | ❌ Нельзя |
|----------|-----------|
| Использовать базовую математику (`Math.sin`, `Math.cos`, `Math.atan2` и т.д.) | Использовать `THREE.Vector3` |
| Писать свои функции для работы с векторами | Использовать `THREE.Quaternion` |
| Создавать свои структуры для хранения данных | Использовать `object.lookAt()` внутри своей функции |
| | Использовать любые готовые классы Three.js для расчётов |

## 🎯 Как проверить правильность

1. Открой приложение (`npm start`)
2. Покрути камеру мышкой — глаз должен **всегда смотреть прямо на камеру**
3. **Включи свою функцию** через интерфейс
4. Если глаз продолжает смотреть на камеру — ✅ всё верно  
   Если глаз смотрит криво — ❌ ошибка в расчётах

## 🎮 Управление в приложении

| Действие | Результат |
|----------|-----------|
| Левая кнопка мыши + движение | Вращение камеры вокруг сцены |
| Колёсико мыши | Приближение / отдаление |
| Чекбокс «Смотреть на камеру» | Включает/выключает вращение глаза |
| Переключатель | Выбор между `lookAt` (встроенная) и `lookAtCustom` (твоя функция) |

## 📍 Исходные данные

- **Позиция глаза:** `(0, 3, 0)`
- **Позиция камеры:** `(5, 5, 5)` — меняется через OrbitControls
- **Координатная система:** Three.js (Y — вверх, Z — вперёд)
- **Углы вращения:** радианы, порядок XYZ (Euler)
